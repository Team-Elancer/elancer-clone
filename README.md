## 참고사항 feature

`REST API를 통한 웹 개발 - React + Spring Boot`

- 무한 스크롤 구현
- 스켈레톤 UI 구현
- 로딩 스피너 구현
- 회원가입/로그인 및 Access/Refresh Token 처리 구현
- 이미지 처리 구현
- 문의사항 CRUD 구현
- 프리랜서 프로필 정보 CRUD 구현
- 기업 정보 CRUD 구현
- 인터뷰 조정 구현
- 찜 기능 구현
- 멀티 필터 기능 구현
- Swiper 라이브러리 사용해 페이징 처리
- Daum API를 사용해 프리랜서 주소 처리
- ETC
